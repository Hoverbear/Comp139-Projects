A Simple List ADT (modified)

Background

In the previous lab you implemented a simple ListADT using a dynamically allocated linear linked list. In this lab you will modify the simple ListADT so that:
  1. Arrays are used to implement the linked list.  See the 'Additional information' section (below) for more information.
  2. The ListADT provides an iterator()  method for use by the client.
  3. The data is sorted using a Quicksort algorithm.

Problem Statement

 Modify your implementation of SimpleStringList class (call it SimpleComparableList) but do not deviate from the specification as defined in the html documentation. Your modifications will include changing the way you store the list items, changing the algorithm that you use to sort the list, and providing an iterator method.

Download Comp139Lab3.zip and extract all files. The zip file contains the lab instructions (this file) as well as :

  •  documentation (in the Doc folder)
  •  the required class files (for the driver)

Once the files have been extracted from the .zip file, you can open the project using NetBeans. Go to menu item "File" and then "Open Project...". Navigate to 'NetBeansProjectFiles', which is a directory that you just extracted. After the project has openend, you can add your source files to the 'source' directory.

The main program which can be used to test your code will be similar to the one used in the previous lab. To use this main program, refer to the instructions in previous lab. 

Additional Information

    • Use arrays to implement the list. To do this you will need the following arrays:
      • an array of Comparable to store the data
      • an array of  integers to represent the linked structure. The integers are indexes into the data array.
      • an array of integers to represent a list of free (unused) elements in the data array.

        Look at this diagram which is an example of how to use these three arrays

    • Sort the list using a Quicksort algorithm.
    • You may write and use your own interface (main program) but in the end, your SimpleComparableList must work with my main program.
    • Write an inner class which implements the Iterator interface. You can instantiate this class when the iterator() method is called.
