This figure shows a one-way linked list of strings implemented in an array. To follow the list, begin with listHead.  The value of listHead is the index of the data array that contains the first data item (a string)  in the list. The corresponding element in the links array contains the index of the next item in the data array. Follow the links in this fashion until you reach an index value of –1, indicating end of list.

freelistHead and the free list array represent a linked list of the empty (unused) elements of the data array.
