A Simple List ADT

Background

A simple List ADT will allow a client program to store, remove and retrieve data objects. This ADT should make it convenient for the client to refer to an item by its position in the list and to sort the list if necessary. In this lab you will create a simple List ADT that conforms to specifications that are provided below.
 

Problem Statement

Read the specifications for the SimpleStringList ADT as defined in the html documentation for SimpleStringList. Write the implementation that conforms to these specifications and test your work with the interface program called COMP139Lab2Driver. Your implementation must be done using a one-way linked linear list. Details about the implementation of the SimpleStringList class and the node class for the linked list will be discussed in the lectures..

The main program which can be used to test your code is called COMP139lab2Driver.class. To use this main program, copy it and all (12) of its inner classes  to the folder that contains your SimpleStringList.class file. Then run COMP139Lab2Driver. This main program will look for and use a class file named SimpleStringList.class. 

You can try a working version of this lab by downloading and running the following jar file: 
Comp139Lab2.jar


To run a  jar file:

  1. Start a Command Prompt.   (Start/Programs/Accessories/Command Prompt)
  2. From the command line, navigate to the directory that contains the jar file.
  3. type:  java -jar Comp139Lab2.jar

  All the support files are in a zip file called  Comp139Lab2.zip. This zip file contains a runnable jar file, all the class files that you need to solve the lab, and all documentation including this html page. 

To create a NetBeans project that uses the distributed class files, first create a directory structure that looks something like this:

Then,  you need to tell NetBeans where the class filse are, and which one is the main class. Here are the steps:

  1. Right click on the project name and select "properties".
  2. Select "Libraries" from the Categories list.
  3. Click the button labeled "Add JAR/Folder"
  4. Navigate to and highlite the folder that contains the distributed class files. This will be 'driverclasses' if you followed the above structure.
  5. Click "Open" then "OK"
To set the main class:
  1. Right click on the project name and select "properties".
  2. Select "Run" from the Categories list.
  3. Type "COMP139Lab2Driver" in the Main Class edit box
  4. Click "OK"

Additional Information

    • Use a one-way sequential linked list to implement the list.
    • You must provide a method that sorts the list. (see the specs) You may use any algorithm you wish but a simple solution is to create a new list by inserting each element from the old list into the new list while maintaing ascending alphabetic order during each insert. Once the new list has been created (or while it is being created) the old list nodes can be destroyed.
    • You may write and use your own interface (main program) to test your code, but in the end, your SimpleStringList must work with my main program. Do not modify my main program (COMP139Lab2Driver).
